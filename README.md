# 🌱 About AGRISMART AI

AGRISMART AI is an intelligent agriculture support system designed to help farmers identify crop diseases using AI-powered image analysis. Farmers can upload an image of an infected leaf, and the system detects the disease severity along with possible recommendations.

The project focuses on reducing farmers’ mental load, improving crop monitoring, and enabling faster disease diagnosis through artificial intelligence and modern web technologies.
