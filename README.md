# 🌱 About AGRISMART AI

AGRISMART AI is an AI-powered agriculture support platform designed to help farmers identify crop diseases through image analysis. Farmers can upload images of infected leaves, and the system predicts the disease along with severity insights and smart recommendations.

The project aims to reduce farmers’ mental load, improve productivity, and enable fast disease diagnosis using Artificial Intelligence and modern web technologies.

# ⚡ Startup Guide

## 🔹 Prerequisites

Before starting the project, install:

- Node.js
- npm
- Git
- VS Code

Check versions:

```bash
node -v
npm -v
git --version
```

---

# 🚀 Project Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/agrismart-ai.git
```

---

## 2️⃣ Move Into Project Directory

```bash
cd agrismart-ai
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Configure Environment Variables

Create a `.env.local` file in the root directory.

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

---

## 5️⃣ Run Development Server

```bash
npm run dev
```

---

## 6️⃣ Open Application

Visit:

```bash
http://localhost:3000
```
